# Overview

These core components are included in Mirage:

-   [Network Animator](/docs/components/network-animator)
    The Network Animator component allows you to synchronize animation states for networked objects. It synchronizes state and parameters from an Animator Controller.

-   [NetworkPositionSync](/docs/components/network-position-sync)
    The Network Position Sync component synchronizes the position of game objects across the network.

-   [DisplayMetrics](/docs/components/display-metrics)
    The Display Metrics component shows network metrics.

-   [HeadlessAutoStart](/docs/components/headless-auto-start)
    The Headless Auto Start component automatically starts the server in headless mode.

-   [HeadlessFrameLimiter](/docs/components/headless-frame-limiter)
    The Headless Frame Limiter component limits the frame rate in headless mode.

-   [LobbyReady](/docs/components/lobby-ready)
    The Lobby Ready component manages the ready state of players in a lobby.

-   [NetworkManagerGUI](/docs/components/network-manager-gui)
    The Network Manager GUI provides a simple user interface for the Network Manager.

-   [NetworkManagerHud](/docs/components/network-manager-hud)
    The Network Manager HUD is a quick-start tool to help you start building your multiplayer game straight away, without first having to build a user interface for game creation/connection/joining. It allows you to jump straight into your gameplay programming and means you can build your own version of these controls later in your development schedule.

-   [NetworkPingDisplay](/docs/components/network-ping-display)
    Network Ping Display shows the Ping time for clients using OnGUI

-   [NetworkTransform](/docs/components/network-transform)
    The Network Transform component synchronizes the movement and rotation of game objects across the network. Note that the network Transform component only synchronizes spawned networked game objects.

-   [NetworkTransformChild](/docs/components/network-transform-child)
    The Network Transform Child component synchronizes the position and rotation of the child game object of a game object with a Network Transform component.

-   [OnlineOfflineScene](/docs/components/online-offline-scene)
    The Online Offline Scene component manages the online and offline scenes.

-   [ReadyCheck](/docs/components/ready-check)
    The Ready Check component manages the ready state of players.

-   [SyncObjectActive](/docs/components/sync-object-active)
    The Sync Object Active component synchronizes the active state of game objects.
